# SSTI to Remote Code Execution

## Overview

This task demonstrates a structured Vulnerability Assessment and Penetration Testing (VAPT) workflow conducted on TryHackMe lab environment. The engagement focused on identifying and validating a Server-Side Template Injection (SSTI) vulnerability that ultimately led to Remote Code Execution (RCE).

## Target Information

- Target: TryHackMe lab machine

- Hostname: ssti.thm

- Target IP: 10.49.129.173

- Testing Type: Black-box

## Tools Used

- Browser (manual testing)

- Burp Suite (request inspection)

- SSTImap (validation assistance)

- Netcat (listener)

- Python HTTP server (payload hosting)
