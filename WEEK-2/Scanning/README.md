
## Objective
Perform vulnerability scanning on Metasploitable2 to identify exposed services
and potential security weaknesses.

## Tools Used
- Nmap
- OpenVAS
- Nikto

## Workflow Summary
- Identified live host
- Conducted full port and service scan
- Validated exposed services
- Mapped findings to risk levels using CVSS
